# 資料科學的 OSEMN 模式

Mason & Wiggins \(2010\) 定義了資料科學的五個步驟: 🧩 



1. **O**btaining data      抓取
2. **S**crubbing data     清理
3. **E**xploring data       探索
4. **M**odeling data       建模
5. i**N**terpreting data   詮解

簡稱 **OSEMN** model \(發音 _awesome_\)。
