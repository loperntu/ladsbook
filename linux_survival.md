# Linux 指令列存活指令



## 目錄和檔案處理
- pwd（顯示目前工作目錄）
- ls（列出當前目錄下的內容）
- cd（更換工作忙碌）
- cp（拷貝檔案及目錄）
- mv（移動或重新命名檔案）
- mkdir（建立目錄）
- rm（刪除檔案或目錄）

```bash
rm -rf non-empty_dir
```

## 文本處理

- echo

```bash
echo "語言學很重要" > note.txt
```
- cat（依標準輸出顯示檔案內容）
- head
- tail
- more（看檔案內容，滿頁暫停按 space 鍵繼續，按 q 結束）
- less（同上，但允許游標鍵上下捲動對內容進行瀏覽）
